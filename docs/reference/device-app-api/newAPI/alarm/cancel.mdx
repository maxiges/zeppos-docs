# cancel

:::note
Start from API_LEVEL `3.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).
:::

Cancels the set timer, if the timer is set to persist and also cancels the persistence.

:::info
permission code: `device:os.alarm`
:::

## Type

```ts
function cancel(option: Option): Result
```

### Simplified calling method

```ts
function cancel(id: number): Result
```

## Parameters

### Option

| Property | Type     | Required | DefaultValue | Description                    | API_LEVEL |
| :------- | :------- | :------- | :----------- | :----------------------------- | :-------- |
| id       | `number` | Y        | -            | The ID of the alarm to cancel. | 3.0       |

### Result

| Type     | Description                                  |
| :------- | :------------------------------------------- |
| `number` | Returns `0` if successful, otherwise failed. |

## Example

import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

<Tabs>
  <TabItem value="single" label="Cancel Single Alarm">

```js
import { cancel } from '@zos/alarm'

  </TabItem>
  <TabItem value="all" label="Cancel All Alarms">

// Get all alarm IDs for this app and cancel them all

alarmIds.forEach((id) => {

})
```
