# getApp

:::note
Start from API_LEVEL `2.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).
:::

Get the app instance object.

`getApp()` allows you to access the singleton application instance registered via `App()`. This is essential for managing global state and sharing data between different pages of your Mini Program.

## Type

```ts
function getApp(): Result
```

## Explanation

The `App()` constructor registers the lifecycle and global state of your application. Since `getApp()` returns the instance, you can use it to:

- Access `globalData` defined in `App()`.
- Share variables between pages (e.g., user settings, temporary state).
- Invoke methods defined in the `App` instance (though rarely used).

**Important Note:** The properties defined in your `App()` constructor are accessible via the `_options` property of the returned app instance.

## Parameters

### Result

| Property  | Type                 | Description           | API_LEVEL |
| --------- | -------------------- | --------------------- | --------- |
| \_options | <code>Options</code> | app instance property | 2.0       |

### Options

| Property   | Type                | Description                           | API_LEVEL |
| ---------- | ------------------- | ------------------------------------- | --------- |
| globalData | <code>object</code> | mounted data objects on app instances | 2.0       |

## Usage Examples

| Scenario               | Code Pattern                                               | Description                   |
| :--------------------- | :--------------------------------------------------------- | :---------------------------- |
| **Basic Access**       | `const app = getApp();`                                    | Retrieve the app instance.    |
| **Read Global Data**   | `const data = getApp()._options.globalData.key;`           | Get value from `globalData`.  |
| **Update Global Data** | `getApp()._options.globalData.key = 'new';`                | Modify value in `globalData`. |
| **Dynamic Key**        | `const k = 'user'; getApp()._options.globalData[k] = val;` | Dynamic access.               |

import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

<Tabs>

```js title="app.js"
App({
  globalData: {
    userName: 'Runner',
    theme: 'dark'
  },
  onCreate() {
    console.log('App Created')
  }
})
```

```js title="page/index.js"
// Retrieving data
const app = getApp()
console.log(app._options.globalData.userName) // Output: 'Runner'

// Updating data from another page
app._options.globalData.theme = 'light'
console.log(app._options.globalData.theme) // Output: 'light'
```
